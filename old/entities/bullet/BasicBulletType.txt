타입 이름: BasicBulletType
포함 대상: BulletType
타입 종류: 클래스

설명:
    포탑과 유닛에서 발사되는 대부분의 탄약 기반 탄환을 위한 확장형 BulletType.

일반 필드
    backColor: {Color} [기본값: f9c27a][후면 색]
    frontColor: {Color} [기본값: fff8e8][전면 색]
    mixColorFrom: {Color} [기본값: ffffff00][
    mixColorTo: {Color} [기본값: ffffff00][
    width: float [기본값: 5][탄환 너비, 가로]
    height: float [기본값: 7][탄환 높이, 세로]
    shrinkX: float [기본값: 0][
    shrinkY: float [기본값: 0.5][
    spin: float [기본값: 0][
    sprite: String [기본값: ][스프라이트 이름]

특이사항(기본값 필드)
    sprite: String [기본값: bullet][스프라이트 이름]
    speed: float [기본값: 1][탄환 속도]
    damage: float [기본값: 1][탄환 데미지]

특이사항(스프라이트)
    이름-back: 앞부분 스프라이트
    이름: 앞부분 스프라이트
