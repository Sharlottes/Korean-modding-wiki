타입 이름: ItemStack
포함 대상: X
타입 종류: 클래스

설명:
    문자열: 아이템/수량
    객체:
        item: 아이템
        amount: 수량


예시:
    consumes: {
      items: {
        items: [
          spore-pod/1
          copper/5
        ]
        item: spore-pod
        amount: 5
        item: copper
        amount: 1
      }
    }
